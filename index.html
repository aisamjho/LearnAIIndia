About Learn AI India
Learn AI India is a free learning platform where anyone can understand Artificial Intelligence in simple everyday English. No technical background needed. Learn from home, at your own pace, completely free — forever.
🔍 How to Get on Google Search
Step 1: Go to search.google.com/search-console → Sign in with Google

Step 2: Click "Add Property" → Choose "URL prefix" → Enter:
https://aisamjho.github.io/aisamjho

Step 3: Verify ownership — Download the HTML file → upload to your GitHub repo → click Verify

Step 4: Go to Sitemaps → enter your URL → Submit

Step 5: Wait 3–7 days → search "Learn AI India Abhishek Tiwari" on Google → your app appears! ✅
📱 Install as App on Any Device
This app is a PWA (Progressive Web App) — users can install it on their home screen and it looks exactly like a real app with a 🧠 icon, no browser bar!

📱 Android: Open in Chrome → tap ⋮ menu → "Add to Home screen" → Install

🍎 iPhone: Open in Safari → tap Share button → "Add to Home Screen" → Add

💻 Desktop Chrome: Look for ⊕ icon in the address bar → click it → Install
